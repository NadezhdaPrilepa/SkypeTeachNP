Значения полей по умолчанию:

- По умолчанию значение всех полей объектов = null
- Для примитивных типов int, double=0
- boolean по умолчанию=false

Блоки инициализацииЖ

- При вызове конструктора, сначала проходит инциализация полей, и лишь потом выполняется
логика конструктора.
- Блоки инциализации(для static и не только) вызываются ПЕРЕД вызовом конструктора.
- Инициализация полей происходит по порядку строк кода, где поля инициализированы,
а не объявлены
- Для констант необходима явная инициализация, по умолчанию ее нет, так же константу
можно инициализировать только единажды.
- Static блок инициализации запускается в момент загрузки класса, а не создания объекта,
поэтому, он вызывается самым первым.
- Константы объекта, кроме явного объявления и блока инициализации, можно проинициализировать
в конструкторе.
- Static константу в конструторе нельзя проинициализировать, т.к. static блок инициализации
запускается в момент загрузки класса, а не создания объекта.
- Может быть несколько блоков инициализации в рамках одного класса

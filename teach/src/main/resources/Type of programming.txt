Типы программирования:
1# Структурное(язык С)
В программировании используются неупорядоченные и несвязанные между собой функции

2# Функциональное (язык и List,Scala)
В программировании самое главное-функция, которая может быть параметром или результатом функции

3# Логическое(язык Prolog)
Все программа, состоит из фактов и правил

4# ООП (языки C++, JAVA)
Программа состоит из взаимодействия объектов, их св-в и методов
ООП основана на 3 парадигмах
- Инкапсуляция
Сосредоточение всех св-в и методов в рамках одного объекта(для JAVA класса)
Все св-ва(private) объекта необходимо передавать "наружу" методами(public)

- Наследование
Расширение объекта. Главный вопрос: Является? Если ответ на вопрос положителен, то наследование возможно

- Полиморфизм
Наделение одного и того же имени разными возможностями (Для JAVA имя=метод)
